# SafarSend Website

Official website for SafarSend - Fast, secure, and reliable file transfer app.

## About

This is a single-page website showcasing the SafarSend mobile application. The website includes:

- **Hero Section** with an interactive slider displaying app screenshots
- **Download Buttons** for Google Play and App Store
- **About Section** describing the app's key features
- **Timeline** showing app updates and version history
- **Footer** with social media links and legal documents

## Features

- Fully responsive design (mobile, tablet, and desktop)
- Interactive image slider with auto-play and manual controls
- Smooth scroll navigation
- Modern gradient design with animations
- Touch/swipe support for mobile devices
- Placeholder images when assets are not available

## Structure

- `index.html` - Main HTML file
- `styles.css` - All styling and responsive design
- `script.js` - JavaScript for slider functionality and animations
- `assets/` - Directory for images (logo and screenshots)

## Usage

Simply open `index.html` in a web browser. No build process or server required.

### Adding Your Own Assets

Place your images in the `assets/` directory:
- `logo.svg` - Your app logo
- `screenshot1.png`, `screenshot2.png`, `screenshot3.png` - App screenshots (300x600px recommended)

### Customization

1. **Update Links**: Edit the download button URLs in `index.html` to point to your actual app store pages
2. **Social Media**: Update social media links in the footer
3. **Content**: Modify text content in `index.html` to match your app's features and updates
4. **Colors**: Adjust CSS variables in `styles.css` (`:root` section) to match your brand

## Browser Support

Works on all modern browsers including:
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)
